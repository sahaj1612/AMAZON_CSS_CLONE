# Amazon CSS Clone

Amazon CSS Clone is a frontend practice project that recreates the look and layout of the Amazon homepage using only HTML and CSS.

## Tech Stack

- HTML
- CSS
- Font Awesome

## Highlights

- Navigation bar layout
- Search section styling
- Hero banner area
- Product grid cards
- Footer sections

## Run Locally

Open `index.html` in any browser.

## Note

This project is built for UI practice and is intended as a static clone.
